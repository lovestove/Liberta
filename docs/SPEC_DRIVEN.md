# SPEC_DRIVEN: Liberta (Android VPN Service)

## 1. Концепция проекта
**Liberta** — это революционный VPN-сервис для Android, созданный для обеспечения абсолютной цифровой свободы. Проект сочетает в себе мощь ядра `sing-box`, передовые технологии обфускации и премиальный дизайн в стиле **Liquid Glass**.

Приложение полностью бесплатно и работает на базе открытых подписок, автоматически выбирая самый быстрый сервер для пользователя.

## 2. Технологический стек

### 2.1. Core (Ядро)
- **Язык**: Go (Golang) 1.22+.
- **Библиотека**: `sing-box` (универсальный прокси-клиент).
- **Сборка**: `gomobile` (компиляция в `.aar` для Android).
- **Протоколы**: VLESS (Reality, ShadowTLS), Trojan, TUIC v5, Hysteria2.

### 2.2. Android App (Клиент)
- **Язык**: Kotlin.
- **UI**: Jetpack Compose + Canvas/Skia для сложных анимаций.
- **Архитектура**: MVVM + Clean Architecture.
- **DI**: Hilt.
- **Хранилище**: Jetpack DataStore.

## 3. Функциональные возможности

### 3.1. Режимы работы (Тумблер на главном экране)
Приложение переключается между двумя глобальными наборами правил:
1.  **Белые списки**: Доступ к ресурсам через прокси только для определенных доменов/IP.
    - [Источник](https://raw.githubusercontent.com/igareck/vpn-configs-for-russia/refs/heads/main/WHITE-CIDR-RU-all.txt)
2.  **Черные списки**: Весь трафик через VPN, кроме исключенных ресурсов.
    - [Источник](https://raw.githubusercontent.com/igareck/vpn-configs-for-russia/refs/heads/main/BLACK_VLESS_RUS.txt)

### 3.2. Liberta Labs (Экспериментальные функции в настройках)
Раздел в настройках, позволяющий пользователю активировать инновационные технологии:

- **Neural Stealth (ИИ-маскировка)**:
    - *Что делает*: Использует локальную модель TinyML для анализа паттернов трафика и изменения структуры VPN-пакетов (padding, timing) под вид обычного HTTPS-трафика.
    - *Цель*: Обход самых жестких систем DPI.
- **Sovereign Mesh (P2P Мост)**:
    - *Что делает*: Позволяет устройству работать в режиме «моста», помогая другим пользователям обходить блокировки через децентрализованную сеть.
    - *Цель*: Создание неубиваемой инфраструктуры.
- **Quantum Shield (Пост-квантовая защита)**:
    - *Что делает*: Активирует алгоритмы ML-KEM (Kyber) для защиты данных от расшифровки будущими квантовыми компьютерами.
- **Generative Heartbeat (Живой фон)**:
    - *Что делает*: Включает генеративный фон на главном экране, который визуализирует поток данных (скорость/пинг) через частицы и градиенты.

## 4. Дизайн и UX (Liquid Glass Style)

### 4.1. Визуальные характеристики
- **Эфеекты**: Стекломорфизм, размытие фона (Blur), динамические тени.
- **Floating UI**: Все элементы отделены от фона, создавая ощущение легкости.
- **Цвета**: Глубокий серый в простое, переходящий в яркие радиальные градиенты (Electric Blue, Radiant Purple) при подключении.

### 4.2. Основные элементы интерфейса
- **Заголовок**: Стильный логотип "Liberta" в верхней части.
- **Кнопка QR (Top-Right)**: Мгновенная генерация QR-кода для шеринга приложения.
- **Кнопка Settings (Top-Left)**: Доступ к Liberta Labs и системным настройкам.
- **Центральная кнопка**: Крупный круглый переключатель. При нажатии — эффект «жидкой волны», расходящейся по всему экрану.
- **Нижний переключатель**: Элегантный Slider-тумблер для выбора между Белыми и Черными списками.

## 5. План реализации для Хакатона

1.  **Этап 1: Core**: Настройка `sing-box` конфигов и компиляция базового Go-ядра.
2.  **Этап 2: UI Foundation**: Создание дизайн-системы Liquid Glass в Compose.
3.  **Этап 3: Labs Integration**: Реализация переключателей в настройках и их связи с ядром.
4.  **Этап 4: Animation Polish**: Отработка «жидкой волны» и генеративного фона.
5.  **Этап 5: Delivery**: Сборка финального .apk и подготовка презентации.
