# SPEC_DRIVEN: Liberta (Android VPN Service)

## 1. Концепция проекта
**Liberta** — это интеллектуальная сетевая оболочка нового поколения для Android. Это не просто VPN, а самозалечивающаяся нейронная сеть, обеспечивающая мгновенный и безопасный доступ к глобальному интернету через премиальный интерфейс в стиле **Liquid Glass**.

## 2. Технологический стек

### 2.1. Core (Ядро)
- **Язык**: Go (Golang) 1.22+.
- **Библиотека**: `sing-box`.
- **Спецификация**: **Zero-Downtime Engine**. Параллельная проверка (Parallel Probing) доступности серверов и мгновенное переключение при деградации канала.
- **Сборка**: `gomobile`.

### 2.2. Android App (Клиент)
- **UI**: Jetpack Compose + GLSL Shaders для **Adaptive Living UI**.
- **Архитектура**: MVVM + Clean.
- **Хранилище**: DataStore.

## 3. Функциональные возможности

### 3.1. Режимы работы (Тумблер на главном экране)
1.  **Белые списки**: Интеллектуальный обход блокировок только для целевых ресурсов.
2.  **Черные списки**: Полная защита всего трафика.

### 3.2. Самозалечивание (Zero-Downtime Switching)
- **Link Monitoring**: Постоянный мониторинг качества соединения (RTT, Packet Loss).
- **Silent Re-routing**: При падении сервера приложение автоматически и бесшовно переключает пользователя на следующий лучший узел из подписки без разрыва системного VPN-соединения.
- **Parallel Probing**: Использование алгоритма "Happy Eyeballs" для одновременной проверки нескольких серверов и выбора самого быстрого в реальном времени.

### 3.3. Liberta Labs (Настройки)
- **Neural Stealth**: ИИ-маскировка трафика под обычные HTTPS-сессии.
- **Sovereign Mesh**: Режим P2P-моста для децентрализованной поддержки сети.
- **Quantum Shield**: Пост-квантовое шифрование (ML-KEM).
- **UI Fidelity (Adaptive Heartbeat)**: 
    - *High-Fi*: Максимальная детализация генеративного фона (реакция на тапы, пакеты).
    - *Eco*: Упрощенный визуальный стиль для максимального энергосбережения.

## 4. Дизайн и UX (Liquid Glass Style)

### 4.1. Визуальные характеристики
- **Adaptive Living UI**: Фон приложения «дышит» и живет в такт вашему интернету. Больше данных — активнее визуализация.
- **Эффект активации**: При нажатии "Connect" от центра расходится «жидкая волна», окрашивая серый интерфейс в живые градиенты.

### 4.2. Элементы управления
- **Smart Warm-up**: Предварительная подготовка серверов при открытии приложения для мгновенного старта.
- **QR & Share**: Интегрированная система приглашения друзей через красивые QR-карточки.

## 5. Инновационные "Wow" факторы для жюри
1.  **Invisible Repair**: Визуальная индикация самозалечивания сети (тонкие световые импульсы при авто-переключении сервера).
2.  **Haptic Pulse**: Синхронизация тактильного отклика с сетевым трафиком.
3.  **Predictive Routing**: ИИ-прогноз качества серверов на основе истории подключений.
