# Спецификация проекта Liberta (Android VPN Service)

## 1. Общее описание
**Liberta** — это премиальный, полностью бесплатный VPN-сервис для Android, построенный на базе современного ядра `sing-box`. Основная философия проекта: «Свобода должна быть красивой и доступной». Приложение использует открытые и бесплатные подписки на сервера, обеспечивая пользователю максимально быстрое и надежное соединение через элегантный интерфейс в стиле **Liquid Glass**.

## 2. Технологический стек

### 2.1. Ядро (Core)
- **Язык**: Go (Golang).
- **Библиотека**: `sing-box`.
- **Инструментарий**: `gomobile`. Сборка в `.aar` архив для нативной интеграции в Android Studio.
- **Возможности**: Полная поддержка протоколов VLESS, Reality, ShadowTLS, Trojan, VMess и др. Продвинутая маршрутизация и шифрование.

### 2.2. Клиентское приложение (Android)
- **Язык**: Kotlin.
- **UI-фреймворк**: Jetpack Compose (Modern Declarative UI).
- **Архитектура**: MVVM (Model-View-ViewModel) + Clean Architecture.
- **Инъекция зависимостей**: Hilt.
- **Хранение данных**: Jetpack DataStore (конфигурации и настройки).
- **Сериализация**: Kotlinx.serialization (для работы с JSON конфигами sing-box).

### 2.3. Системный уровень
- **Android SDK**: `VpnService` для создания TUN-интерфейса.
- **Local Tun Stack**: Стек `gVisor` или `lwIP` (встроено в sing-box) для обработки пакетов.

## 3. Функциональные требования

### 3.1. Работа с подписками
Приложение оперирует двумя типами подписок (режимов):
1.  **Белые списки (Allow-list)**: https://raw.githubusercontent.com/igareck/vpn-configs-for-russia/refs/heads/main/WHITE-CIDR-RU-all.txt
2.  **Черные списки (Block-list)**: https://raw.githubusercontent.com/igareck/vpn-configs-for-russia/refs/heads/main/BLACK_VLESS_RUS.txt

**Логика подключения**:
- При смене режима или нажатии кнопки «Подключить», приложение загружает актуальную подписку.
- Выполняется быстрый замер (latency test) всех серверов в подписке.
- Выбирается наиболее быстрый/стабильный сервер и автоматически генерируется `sing-box` конфиг.
- Устанавливается VPN-соединение.

### 3.2. Настройки
- **Управление подписками**:
    - Индикация даты/времени последнего обновления.
    - Переключатель автообновления при запуске.
    - Кнопка принудительного ручного обновления.
- **Системные настройки**:
    - Автозапуск при загрузке устройства.
    - Kill Switch (блокировка трафика при разрыве VPN).
    - Выбор Tun-стека (gVisor/lwIP).
- **Прочее**: Темная/светлая тема (адаптивная под систему).

## 4. Дизайн и UX (Liquid Glass Style)

### 4.1. Визуальный стиль
- **Эстетика**: «Жидкое стекло» (Liquid Glass) с эффектами размытия, глубины и мягких теней.
- **Floating UI**: Все элементы интерфейса представлены в виде «парящих» блоков с выраженным отделением от фона.
- **Типографика**: Использование современных шрифтов (например, *Outfit* или *Inter*) для дорогого и профессионального вида.

### 4.2. Главный экран
- **Верхняя панель**:
    - Слева: Кнопка настроек (иконка-шестеренка в стиле glassmorphism).
    - Центр: Название проекта «Liberta» с градиентным начертанием.
    - Справа: Кнопка QR-кода. При нажатии открывается QR-код приложения с надписью «Поделись с друзьями».
- **Центральная часть**:
    - Крупная интерактивная кнопка включения/выключения VPN.
- **Нижняя панель**:
    - Стилизованный тумблер (Switch) для переключения между «Белыми списками» (слева) и «Черными списками» (справа). Никаких лишних подписей — интуитивно понятный визуальный выбор.

### 4.3. Анимации и спецэффекты
- **Эффект активации**: При нажатии на центральную кнопку от неё во все стороны расходится «жидкая волна» (liquid ripple effect).
- **Фоновые эффекты**: После активации фон становится динамическим — появляются переливающиеся радиальные градиенты и мягкие световые пятна, создающие эффект «ожившего» интерфейса.
- **Плавность**: Все переходы между экранами и состояниями элементов (нажатия, переключения) анимированы максимально мягко.

## 5. Идеи для победы на Хакатоне (Innovation Suite)

Для того чтобы продукт стал эталонным и выделился среди конкурентов, предлагаются следующие инновации:

1.  **Smart Protocol Obfuscation**: Автоматический подбор параметров обфускации (Reality/ShadowTLS) в зависимости от типа блокировок в сети пользователя. Приложение само понимает, когда его «детектят», и меняет стратегию.
2.  **Decentralized Backup Sync**: Если основной источник подписок (GitHub) будет заблокирован, приложение автоматически переключится на резервные зеркала или DNS-over-HTTPS записи для получения новых конфигов.
3.  **Haptic Feedback Engine**: Синхронизация тактильной отдачи (вибрации) с анимацией «жидкой волны». Пользователь буквально чувствует запуск свободы.
4.  **Network Health Dashboard**: Маленький, красивый виджет внутри приложения, показывающий не просто «вкл/выкл», а стабильность канала в реальном времени через красивые графики.
5.  **Battery Saver Core**: Тонкая настройка Go-ядра для работы в режиме минимального энергопотребления (Idle mode), что критично для мобильных устройств.
