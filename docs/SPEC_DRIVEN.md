# MASTER SPEC: Liberta (The Neural Mesh VPN)

## 1. Философия проекта
**Liberta** — это интеллектуальная система цифрового суверенитета для Android. Мы переосмыслили концепцию VPN, превратив её из статического туннеля в **самоорганизующуюся децентрализованную сеть**, способную обходить блокировки через инфраструктуру «белых списков» и взаимопомощь пользователей.

## 2. Технологические Столпы

### 2.1. Sovereign P2P Relay (Миллионы точек входа)
- **Суть**: Превращение каждого смартфона в потенциальный защищенный шлюз.
- **Механизм**: Если прямое соединение с сервером заблокировано, приложение через DHT (Distributed Hash Table) находит «соседа» — другого пользователя Liberta, у которого есть доступ (например, через «белый» IP корпорации или другой регион). Трафик идет через него по зашифрованному P2P-каналу.
- **Социальный контракт**: В Labs доступен тумблер **«Стать Суверенным Узлом»**. Пользователь добровольно выделяет ~5% ресурсов для помощи другим.
- **Геймификация**: Счетчик «Вы помогли X людям обрести свободу» — это мотивирует развивать сеть.

### 2.2. Social Steganography (Обновление через «Белые списки»)
- **Суть**: Получение актуальных конфигов и IP серверов в условиях полной изоляции.
- **Механизм**: Приложение обращается к разрешенным ресурсам (ВКонтакте, Яндекс.Картинки) и скачивает изображения из определенных публичных источников. 
- **Технология**: В пикселях этих изображений с помощью нейро-стеганографии зашифрованы актуальные рабочие конфигурации. Для провайдера это выглядит как обычный просмотр картинок в соцсетях.

### 2.3. Phantom Call (WebRTC маскировка)
- **Механизм**: Энкапсуляция трафика в структуру пакетов видеозвонка (WebRTC). Это позволяет проходить сквозь белые списки под видом самого приоритетного и разрешенного трафика (ВК/Яндекс звонки).

### 2.4. Polymorphic DNA & Neural Stealth
- **Мутирующее ядро**: Логика протокола и структура пакетов меняются каждую минуту для обхода сигнатурного анализа.
- **ИИ-маскировка**: Подстройка трафика под статистический профиль «белого» трафика (YouTube, TikTok).

## 3. Технический стек
- **Core**: Go (Golang) + `sing-box`.
- **Android**: Kotlin + Jetpack Compose + MVVM.
- **UI Style**: **Liquid Glass**. Анимация «жидкой волны» при включении и адаптивный живой фон (Eco/High-Fi).

## 4. Стратегия победы на Хакатоне
1.  **Демонстрация неувизимости**: Показ работы VPN через «белый список» с использованием P2P-реле.
2.  **Эмоциональный UX**: «Живой» интерфейс, реагирующий на трафик, и геймификация Sovereign Node.
3.  **Техническая аргументация**: Детальное описание стеганографии и ASN-кластеризации.
