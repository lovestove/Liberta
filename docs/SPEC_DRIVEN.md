# MASTER SPEC: Liberta (The Neural Mesh VPN)

## 1. Философия проекта
**Liberta** — это интеллектуальная система цифрового суверенитета для Android. Мы переосмыслили концепцию VPN, превратив её из простого туннеля в самозалечивающуюся нейронную сеть, способную противостоять любым современным системам блокировок (DPI) при сохранении премиального пользовательского опыта.

## 2. Технологический фундамент

### 2.1. Core (Golang + sing-box)
- **Zero-Downtime Engine**: Система бесшовного переключения между узлами.
- **Infrastructural Deduplication**: Алгоритм кластеризации 1500+ серверов по ASN и подсетям. Вместо перебора всех IP, приложение пингует инфраструктурные кластеры, сокращая время поиска рабочего узла до <300мс.
- **Parallel Connectivity Racing**: Запуск одновременных рукопожатий (handshakes) с несколькими кандидатами. Первый ответивший сервер с валидным HTTP-ответом становится основным.
- **Post-Quantum Security**: Внедрение алгоритмов ML-KEM (Kyber) для защиты от угроз квантовых вычислений.

### 2.2. Android App (Kotlin + Jetpack Compose)
- **Living UI Engine**: Генеративный фон на базе GLSL шейдеров, визуализирующий «сердцебиение» интернета.
- **Haptic Engine**: Синхронизация тактильного отклика смартфона с сетевой активностью и анимациями «жидкой волны».
- **Homeostasis Power Management**: Адаптивное управление частотой кадров и сетевыми пробами в зависимости от заряда батареи и активности пользователя.

## 3. Инновационные модули (Liberta Labs)

### 3.1. Redundant Shield (Двойной туннель)
- **Selective Duplication**: Дублирование только критических пакетов (DNS, VoIP метаданные, Handshakes) через два независимых канала.
- **Warm Standby**: Поддержание второго «горячего» туннеля для мгновенного подхвата трафика при падении основного.
- **Context-Aware**: Автоматическая активация режима при обнаружении нестабильности канала или во время важных сессий (звонки, игры).

### 3.2. Neural Stealth (ИИ-маскировка)
- **TinyML Obfuscation**: Локальная нейросеть анализирует «белый» трафик (YouTube, Zoom) и подстраивает структуру VPN-пакетов под его статистический профиль.
- **Dynamic SNI Mimicry**: Автоматический подбор TLS SNI на основе анализа локально разрешенных и доверенных ресурсов (банки, госуслуги).

### 3.3. Sovereign Mesh (P2P Режим)
- Децентрализованная поддержка сети: пользователи могут активировать режим «моста» для помощи другим участникам сети в обходе блокировок.

## 4. UI/UX Дизайн (Liquid Glass Style)

### 4.1. Главный экран
- **Adaptive Heartbeat**: Фон, который всегда активен. В режиме *Eco* — мягкие градиенты, в *High-Fi* — детальные частицы и шейдерные волны.
- **Liquid Activation**: Плавная анимация «растекающейся волны» при нажатии на центральную кнопку подключения.
- **Smart Warm-up**: Предварительная проверка серверов при запуске приложения для мгновенного (0.2с) коннекта.

### 4.2. Навигация
- **Labs Section**: Специальный раздел в настройках для активации экспериментальных функций с индикацией их влияния на батарею и данные.
- **QR Sharing**: Генерация эстетичных карточек с QR-кодами для безопасного обмена конфигурациями с друзьями.

## 5. Стратегия победы на Хакатоне
1.  **Демонстрация неубиваемости**: Показ бесшовной работы 4K-стриминга при принудительном отключении серверов (Redundant Shield).
2.  **Технологическое превосходство**: Обоснование алгоритма ASN-кластеризации как способа управления огромными списками серверов (1500+).
3.  **Эстетический шок**: Использование Liquid Glass интерфейса и Haptic Pulse для создания эмоциональной связи с продуктом.
4.  **Проработка мелочей**: Акцент на Homeostasis Engine как доказательство заботы о батарее пользователя.
